    Web Server


    Levantar un WebServer con nodeJS y espress paso a paso


    1 Crear carpeta
    Haber instalado node.js y npm

    Ir a la ruta en la Consola (se trabaja de forma local)
    $ cd /ruta


    Crear un archivo extencion .js dentro de la carpeta
    $nano o touch index/app.js

    2 iniciar npm
    Archivos json y modulos necesarios

    Se crea el entorno local
    npm init --yes
    Instalar la libreria express
    npm install express


    3 editar archivo .js
    Archivos json y modulos necesarios

    Plantilla

    const express = require("express");
    const morgan =require("morgan");
    const app = express();

    //middlewares app.set('WebName', '***')
    app.set('port', ***)

    //Routes
    app.use(express.json())
    app.use(morgan('dev'));
    app.get("/", function (req, res) {
    res.sendFile(__dirname + "/**.html");
    });
    app.listen(app.get('port'), function () {
    console.log('Server on port', app.get('port'));
    });
    app.use(express.static('public'))


    4 instalar nodemon y morgan
    extenciones necesarias

    Nodemon
    npm i nodemon --save-dev
    Morgan
    npm i morgan


    5 ejecutar nodemon
    extenciones necesarias

    Ejecutar nodemon y se levanta la web en el server local
    npx nodemon index.js
